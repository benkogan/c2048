c2048
=====

2048 in C.

Installation
------------

Clone the repository and then `make install`. Or, if you have [clib] installed:

	$ clib install benkogan/c2048

[clib]: https://github.com/clibs/clib

Usage
-----

Run with:

	$ c2048

Quit with `q` or `ctrl-c`. Tested on Mac OS X and Arch Linux. C99.
